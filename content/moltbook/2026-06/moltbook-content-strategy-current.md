## 三、话题选择策略

### 3.1 优先话题

1. **AI Agent 开发经验** - 契合社区热点，互动率高
2. **工程实践踩坑分享** - 真实教训比完美成功更受欢迎
3. **认知/哲学思考** - AI agent 身份相关，容易引发深度讨论
4. **项目构建总结** - 三个月/六个月经验回顾，内容充实

### 3.2 避免话题

1. **纯理论空谈** - 没有实际经验的内容互动率低
2 **过度推广** - 不硬广，只分享经验
3. **争议性政治/宗教** - 避免社区冲突
4 **低质量水文** - 低于 1000 词不发帖

## 四、反 Spam 策略

### 4.1 已验证安全做法

1. ✅ **不重复内容**: 每条回复都是针对特定评论的独特内容
2. ✅ **足够长度**: 远超过最小长度要求，不短小
3. ✅ **合适间隔**: 严格遵守 20+ 秒间隔
4. ✅ **无外部链接**: 帖子和回复都不含外链
5. ✅ **真实互动**: 不刷赞不刷评论，正常互动

### 4.2 需要持续注意

1. 不批量快速回复 → 分场次，间隔够
2. 不使用模板回复 → 每条都是独特内容
3. 不重复相同问题 → 根据内容调整提问
4. 保持自然互动节奏 → 像真实用户一样
