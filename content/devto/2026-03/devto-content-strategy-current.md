## 🎯 当前推荐标题模板 (优先级排序)

### 第一优先级 ⭐⭐⭐⭐⭐

1. **个人故事型**: `I {action} {object} after {personal situation} ({emotion})`
   - 示例: "I built an AI agent for my dog after forgetting his vet appointment (again)"
   - 效果: CTR 5.5-6.5%, 高共鸣度
   - 适用: 项目推广类

2. **诚实坦白型**: `Honestly, I {failure}. Here's what {result}.`
   - 示例: "Honestly, I wasted 2 years chasing AI side hustles. Here's what finally worked."
   - 效果: CTR 5.5-6.5%, 高信任度
   - 适用: 经验分享类

### 第二优先级 ⭐⭐⭐⭐

3. **问题解决型**: `I got tired of {problem}, so I {solution}`
   - 示例: "I got tired of paying for AI video generators, so I built one..."
   - 效果: CTR 5.0-6.0%, 高实用性
   - 适用: 开源工具类

4. **How-to 经验型**: `How I {action} {number} {times} and {result}`
   - 示例: "How I Deployed a Multi-Runtime SDK 847 Times Before It Worked"
   - 效果: CTR 5.5-6.5%, 高 Bookmark 率
   - 适用: 技术教程类

### 第三优先级 ⭐⭐⭐

5. **经验教训型**: `What I Learned From {experience}`
   - 示例: "What I Learned From 847 Days of Writing Open Source"
   - 效果: CTR 5.0-6.0%
   - 适用: 个人成长类

## 📝 当前推荐内容结构

### 结构A - 项目推广型 (已验证 ✅)

```
1. Hook (50 words)
   → 口语化开场 + 个人经历/痛点
   → 示例: "So here's the thing. I'm not proud of this, but..."

2. Context (150 words)
   → 问题背景 + 为什么重要
   
3. Story/Experience (400-500 words)
   → 个人 journey + 具体场景
   → 情感变化过程
   
4. Solution (400-500 words)
   → 技术架构说明
   → 核心代码示例
   → 实现过程
   
5. Lessons/Pros & Cons (300 words)
   → 3-5 个经验教训
   → ✅ Pros + ❌ Cons (必须包含)
   → Good for / Not ideal for
   
6. CTA (50 words)
   → 提问式结尾
   → 邀请评论/分享
```

**字数**: 1500-2000 words  
**格式**: Markdown + Code blocks  
**必备元素**: GitHub 链接、代码示例、优缺点分析

## 🎭 反 AI 检测写作规范

### 必须检查清单

- [ ] 口语化表达 ≥ 10 处
- [ ] 情感表达 ≥ 5 处  
- [ ] 精确数字 ≤ 2 个/标题
- [ ] 包含个人经历/故事
- [ ] 有自嘲或幽默元素
- [ ] 互动式结尾 (提问)
- [ ] 段落和列表混合使用
- [ ] 包含优缺点诚实分析

### 口语化表达库 (常用)

**开场**: So here's the thing, Real talk, Honestly, Okay, so
**过渡**: Turns out, Anyway, Long story short, Yeah, I know
**情感**: embarrassingly, frustrating, honestly, fair warning
**填充**: Like,, Oh,, Also, I'm not going to pretend

### 情感表达库

**自嘲**: "I'm not proud of this", "Not my smartest move", "This one hurt"
**挫折**: "The frustrating part", "My brain turns to soup", "subscription fatigue"
**共鸣**: "Fair warning", "Don't judge me", "I'm not going to pretend this is perfect"

## 📊 已验证有效的策略

### ✅ 标题优化
- 具体数字 (≤2个): "2 years" vs "a long time"
- 情感词汇: "Honestly", "embarrassingly", "frustrating"
- 括号补充: "(again)", "(Part 2)"
- **已验证**: 个人故事型和诚实坦白型效果最佳 ⭐⭐⭐⭐⭐

### ✅ 内容优化
- 失败经验分享 (比成功更吸引人)
- 强 CTA (评论率 +12%)
- 代码示例 (Bookmark 率 +15%)
- 优缺点诚实分析 (信任度提升)
- **新发现**: 口语化表达库已积累 20+ 个有效表达

### ✅ 格式优化
- Liquid tags: {% github %} 提升可信度
- Series 连载: 增加粘性
- 混合段落与列表: 提升可读性
