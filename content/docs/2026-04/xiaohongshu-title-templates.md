## 🎯 使用说明

1. **优先使用 Tier 1 模板** - 经过策略验证的高效果模板

## ✅ Tier 1 推荐模板（主推）

### 踩坑实录类 ⭐⭐⭐⭐⭐

#### 模板 1.1：时间+问题型
```
做{项目名}，我被{技术/工具}坑了{时间}
```
**示例**:
- "做 Trip Agent，我被地图 API 坑了3天"
- "折腾 Capa-Java 开源，花了2个月才搞懂社区规则"
- "MCP 协议刚出就踩坑，记录一下"

**效果验证**: ✅ 已验证高点击率，真实感强

#### 模板 1.2：预期反差型 🔄
```
本来以为{简单预期}，结果{意外情况}
```
**示例**:
- "本来以为调用 API 很简单，结果踩了一堆坑"
- "以为开源就是上传代码，没想到维护这么复杂"
- "觉得 AI 能搞定路线规划，结果发现没那么简单"

**效果验证**: 🔄 待Test-006验证，预期效果良好

#### 模板 1.3：具体数字型
```
{项目名}开发了{时间}，说说真实感受
```
**示例**:
- "Trip Agent 开发了8个月，说说真实感受"
- "Capa-Java 维护了3年，记录一些心得"
- "48小时黑客松，我们是怎么拿下金奖的"

**效果验证**: ✅ 时间维度增强可信度，效果稳定

## 🚫 禁用标题清单（保持不变）

### 绝对禁止的套路词
| 禁用词 | 原因 | 替代方案 |
|--------|------|----------|
| "姐妹们！" | 过度亲密，营销感 | "你们"、"大家" |
| "救命！" | 夸张情绪化 | 直接说问题 |
| "挖到宝了！" | 营销套路 | 直接说发现 |
| "绝绝子！" | 网络流行语，过时 | 具体描述优点 |
| "快冲！" | 推销感强 | 理性分析 |
| "yyds！" | 过度夸张 | 具体说明好的原因 |
| "保姆级教程" | 模板化严重 | "详细记录"、"步骤分享" |
| "有手就会" | 低估读者智商 | "思路分享" |
| "神仙工具" | 虚假夸张 | 工具名+具体功能 |
| "宝藏app" | 营销号用词 | 项目名+特点 |
| "效率翻倍" | 缺乏依据的夸张 | 具体效率提升数据 |
| "不看后悔" | 恐吓式营销 | 直接说内容价值 |

### 避免的情绪化夸张
- "震惊！" → 改为 "发现了一个问题"
- "太香了！" → 改为 "确实好用"
- "吹爆！" → 改为 "推荐试试"
- "闭眼入！" → 改为 "可以考虑"
- "我不允许还有人不知道！" → 改为 "分享一下"
