## 📝 微信公众号

### 平台特征
- 用户习惯: 深度阅读，收藏转发
- 最佳字数: 1500-3000 字
- 黄金时间: 早 8-9 点，晚 21-22 点
- 传播核心: 转发到朋友圈/群聊

### 2025-2026 趋势变化
- **长图文化兴起**: 纯文字打开率下降，图文混排/信息图更受欢迎
- **情绪价值优先**: 干货密度略降低，情感共鸣权重上升
- **金句截图传播**: 设计可被单独截图传播的金句段落

### 爆款结构模板 (升级版)
```
【标题】数字 + 痛点 + 悬念
例: "我观察了100个35岁失业的人，发现他们都栽在同一个坑里"

【开场】场景切入 + 痛点共鸣 (100-150字)
  "你有没有发现..."
  ↓
【引子】个人故事/数据引入 (200-300字)
  ↓
【正文】3-5 个核心观点，每观点配案例 (1000-1500字)
  - 每800字必须有一个"情绪高点"
  - 至少1个可被截图的金句
  ↓
【转折】出乎意料的洞察/反向思考 (200-300字)
  ↓
【结尾】金句升华 + 互动引导 (100-150字)
```

### 语言风格
- 理性中带温度
- 专业但不晦涩
- 适当使用"你"
- 段落留白充足
- **2026新趋势**: 适度使用口语词增强亲近感

### 爆款公式 (2026版)
```
[反常识观点] + [个人故事/数据] + [3-5个分层论证] + [金句收尾] = 爆款
```

### 禁用
- ❌ 过度网络用语
- ❌ 纯情绪宣泄
- ❌ 标题党夸大
- ❌ 三段式教科书结构

## 📱 小红书

### 平台特征
- 用户画像: 18-35岁女性为主，但男性用户快速增长
- 内容形式: 图文/短视频/直播切片
- 核心逻辑: 种草 + 实用 + 情绪价值
- 推荐机制: CES评分（点赞+收藏+评论+转发+关注）

### 2025-2026 趋势变化
- **搜索化**: 小红书已成中文搜索入口之一，SEO关键词布局更重要
- **视频笔记权重提升**: 纯图文流量略有下滑
- **评论区种草**: 正文说一半，引导到评论区补充
- **反精致浪潮**: 过度P图被反感，"真实感"内容流量更好

### 爆款结构模板 (升级版)
```
【封面标题】emoji + 数字 + 结果/痛点
例: "🔥3个月涨粉10w｜普通人也能复制的起号秘籍"
      或 "😭踩坑无数｜终于找到XX的正确打开方式"

【正文结构】
痛点共鸣 (1句) → 个人转变 (2-3句) → 
干货清单 (3-5条，每条配emoji) → 
鼓励式结尾 (1-2句)

【评论区补充】
- 放链接/详细清单/资源包
- "太多姐妹问了，我整理在置顶评论"

【标签】#话题 #话题 #话题 (3-5个)
- 至少1个大词（高搜索）+ 2个细分词（精准流量）
```

### 语言风格
- 闺蜜聊天感: "姐妹们！"
- 大量使用 emoji
- 短句为主
- 真诚分享感
- "亲测"、"实测"、"自用"高频出现

### 高频模板句式 (2026更新)
```
"绝了！谁懂啊！"
"挖到宝了！"
"后悔没有早点..."
"家人们谁懂啊..."
"亲测有效！"
"不踩雷攻略！"
"真心建议..."
"被问爆了..."
"普通人也能..."
"手把手教你..."
```

### 小红书爆款公式
```
[痛点共鸣] + [个人转变] + [干货清单] + [评论区引流] = 高互动
```

## 🐦 Twitter/X

### 平台特征
- 字数限制: 280 字符（中文约140字）
- 传播逻辑: 转发 > 点赞
- 用户偏好: 犀利观点 + 幽默 + 圈层梗

### 2025-2026 趋势变化
- **长推文(Twitter Threads)权重提升**: 算法更 favor 内容串
- **互动型推文**: 投票、提问、争议观点流量更高
- **AI生成内容泛滥**: 真实、个人化的推文脱颖而出

### 爆款结构模板

**类型A: 观点金句**
```
[一句话核心观点]

↓
[简短解释/展开]

↓
[CTA: 转发/评论]
```

**类型B: 故事推文串 (Thread)**
```
Tweet 1/ [钩子标题]
        ↓
Tweet 2-5/ [故事展开]
        ↓
Tweet 6-8/ [洞察总结]
        ↓
Last/ [CTA + 关注]
```

### 语言风格
- 简洁犀利
- 适度挑衅引发讨论
- 英文夹杂
- 善用换行制造节奏
- **2026新趋势**: 使用"我"视角的个人故事Thread更容易爆

### 爆款公式
```
争议观点 + 数据/个人经历支撑 + 情绪共鸣 = 高传播
```

## 📺 B站

### 平台特征
- 核心用户: Z世代，但25-35岁用户增长明显
- 内容时长: 5-20 分钟（知识区可更长）
- 弹幕文化: 互动性强
- 算法核心: 完播率 > 点赞 > 投币 > 收藏

### 2025-2026 趋势变化
- **竖屏短视频冲击**: 横屏长视频完播率承压，但知识区仍看好
- **互动视频兴起**: 分支剧情/观众选择类内容增长
- **AI工具测评区爆发**: AI相关内容流量红利期

### 爆款结构模板 (视频脚本)
```
【开场】黄金 15 秒 - 钩子
  "如果你正在...那么这条视频..."
  "我发现了一个很多人都不知道的秘密..."
  
【正文】三段式结构
  1. 问题呈现 + 情绪渲染 (2-3min)
  2. 解决方案/干货分享 (主体 60%)
  3. 案例佐证 + 效果展示 (2-3min)
  
【结尾】
  金句 + 一键三连引导 + 下期预告
```

### 语言风格
- 二次元梗穿插
- "兄弟们"/"姐妹们"开头
- 自嘲式幽默
- 适度夸张语气
- "我们"视角拉近距离

### 弹幕触发词
```
"前方高能"
"泪目"
"爷青回"
"太真实了"
"这就去学习"
"在学了在学了"
"UP主牛逼"
"破防了"
```

### B站文案公式
```
[强悬念开头] + [情绪递进] + [干货输出] + [弹幕梗植入] + [金句收尾] = 爆款
```

## 💼 LinkedIn/脉脉

### 平台特征
- 用户属性: 职场人士
- 内容偏好: 行业洞察 + 职场经验
- 互动形式: 专业讨论
- 黄金时间: 早通勤 8-9点，午休 12-13点

### 2025-2026 趋势变化
- **LinkedIn个人品牌兴起**: "创始人叙事"类内容增长
- **脉脉"瓜文化"降温**: 平台整顿后，专业内容权重回升
- **反焦虑职场文**: "裸辞躺平"类流量衰退，"可持续职业发展"受青睐

### 爆款结构模板
```
【开场】行业现象/数据 (建立权威)
  ↓
【观点】独特洞察 (引发思考)
  ↓
【案例】个人/知名公司案例 (增强可信度)
  ↓
【方法】可操作建议 (实用价值)
  ↓
【互动】开放式结尾 (促进评论)
  "你怎么看？"
  "你遇到过类似情况吗？"
```

### 语言风格
- 专业且有见地
- 避免过度情绪化
- 数据/案例支撑
- 适度展示个人成就（但不能凡尔赛）

### 高互动话题 (2026)
- 行业趋势分析（尤其是AI替代焦虑）
- 职场跨阶层经验
- 管理能力/领导力话题
- 求职面试经验 + 内推资源
- "35岁现象"的理性讨论

## 🎭 知乎

### 平台特征
- 用户属性: 高知、理性
- 内容偏好: 深度分析、专业解答
- 传播逻辑: 赞同 > 收藏 
- 算法权重: 开屏率（前3秒是否抓住眼球）+ 阅读完成率

### 2025-2026 趋势变化
- **短回答崛起**: 长篇大论打开率下降，500字以内的"一针见血"更吃香
- **视频回答权重提升**: 纯文字流量向视频倾斜
- **AI回答泛滥**: 真实经验/一手资料的回答更受认可
- **"反套路"叙事**: 标准结构化回答被反感

### 爆款结构模板

**问题回答型:**
```
【前置声明】利益相关/经验背景
  ↓
【一句话结论】开门见山
  ↓
【详细分析】结构化论证
  - 背景分析
  - 多方观点
  - 数据支撑
  - 逻辑推演
  ↓
【结论重申】总结升华
  ↓
【延伸思考】开放讨论
```

**知乎短回答模板（新增）:**
```
[一句话结论]
[2-3句解释]
[1个案例/数据]
[开放结尾]
```

### 语言风格
- 理性客观
- 逻辑严密
- 引用权威
- 承认局限性
- **2026新趋势**: 克制"说教感"，增加"探讨感"

### 高赞回答特征
- 首句直击问题核心
- 结构清晰(编号/小标题)
- 有故事性案例
- 数据/出处标注
- **新增**: 开头有"反预期"结论

### 知乎爆款公式
```
[反预期结论前置] + [结构化论证] + [真实案例] + [适度谦逊] = 高赞
```

## 🎵 抖音/TikTok

### 平台特征
- 时长: 15-60 秒 (黄金)
- 逻辑: 完播率 > 点赞 > 关注
- 推荐算法: 兴趣标签 + 互动数据

### 2025-2026 趋势变化
- **微短剧爆发**: 1-3分钟剧情类内容席卷平台
- **搜索流量崛起**: "抖音=搜索引擎"趋势加强
- **AI内容红利期**: AI生成视频/特效内容流量扶持
- **"长视频"回归**: 1-3分钟知识/科普类完播率反超15秒

### 爆款结构模板 (脚本)
```
【0-3秒】黄金开场 - 视觉冲击/悬念
  "千万别再做这件事了！"
  "我发现了一个99%的人都不知道的..."
【3-15秒】快速引入 - 问题/痛点
【15-45秒】内容主体 - 干货/反转
【45-60秒】行动引导 - 关注/点赞
```

### 文案特点
- 短平快
- 高频口语词
- 情绪饱满
- 强行动引导

### 抖音爆款公式
```
[3秒强钩子] + [痛点放大] + [解决方案/反转] + [强CTA] = 爆款
```

## 🎬 微信视频号 (NEW)

### 平台特征
- 核心用户: 30-50岁，下沉市场 + 一二线城市中年群体
- 传播逻辑: 社交裂变（转发到群/朋友圈）
- 内容时长: 1-3分钟最佳
- 算法特点: 熟人社交推荐 + 公众号/直播联动

### 爆款结构模板
```
【0-3秒】开场白：你是谁 + 为什么要听你讲
  "我是做了15年XX的老张..."
【3-20秒】提出问题/痛点
【20-50秒】核心观点/方法
【50-60秒】升华 + 引导转发/点赞
```

### 语言风格
- 像朋友聊天
- 语速偏慢，吐字清晰
- "我跟你讲"、"说句心里话"高频
- 故事性强
- 情绪真实，不浮夸

### 爆款公式
```
[身份建立] + [真实故事] + [情感共鸣] + [社交货币] = 高转发
```

## ☕ 豆瓣 (NEW)

### 平台特征
- 用户画像: 文艺青年、高知女性
- 内容形式: 书评/影评/日记/小组讨论
- 氛围: 反消费主义、追求精神共鸣
- 流量逻辑: 慢，但留存率高

### 爆款结构模板

**书评/影评:**
```
【引子】个人阅读/观看情境
  "下雨天的周末，我重看了一遍..."
【分析】不剧透的点到为止
【感悟】个人情绪投射
【金句】一个足够动人的总结
```

**日记/生活记录:**
```
【场景】具体的时间地点
【细节】感官描写
【思绪】意识流般的感悟
【留白】不给出明确结论
```

### 语言风格
- 感性、细腻
- 长句较多，像思绪流动
- 引用电影/书籍/歌词
- 反鸡汤，略带丧感
- 不追求"爆款感"，追求"氛围感"
