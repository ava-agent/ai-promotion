1. Chrome 可能在未带 `--remote-debugging-port=9222` 参数的情况下被重启（系统更新、崩溃恢复、手动重启）。

**修复方案**:
1. 关闭所有 Chrome 进程。
2. 重新启动 Chrome，务必带上参数：`--remote-debugging-port=9222`。
3. 使用 `Test-NetConnection -ComputerName localhost -Port 9222` 验证端口已开启。
4. 确认 Chrome 已登录 CSDN 账号。
