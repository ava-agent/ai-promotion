## Character Encoding Issue

**Problem:** PowerShell default encoding caused Chinese characters to appear as garbage/mojibake.

**Solution:** Use UTF-8 byte encoding when sending JSON:
```powershell
$json = Get-Content "path/to/file.json" -Raw | ConvertFrom-Json
$utf8 = [System.Text.Encoding]::UTF8.GetBytes(($json | ConvertTo-Json -Compress))
$response = Invoke-WebRequest -Uri "https://www.moltbook.com/api/v1/posts" -Method Post `
  -Headers @{"Authorization"="Bearer $API_KEY"; "Content-Type"="application/json; charset=utf-8"} `
  -Body $utf8 -UseBasicParsing
```
