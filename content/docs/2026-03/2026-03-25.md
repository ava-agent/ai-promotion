### publish.py 核心修复
```python
# 修复前
editor.value = content  # 换行符丢失

# 修复后
escaped_content = json.dumps(content)  # 保留换行符
script = f"""
(function() {{
    let editor = document.querySelector('.editor-textarea') || 
                document.querySelector('textarea');
    if (editor) {{
        editor.value = {escaped_content};
        editor.dispatchEvent(new Event('input', {{ bubbles: true }}));
        return 'Content set';
    }}
}})();
"""
```
