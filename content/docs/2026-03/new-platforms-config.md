##### 2. Access Token 获取机制

**接口地址**: `GET https://api.weixin.qq.com/cgi-bin/token`

**请求参数**:
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| grant_type | string | 是 | 固定值 `client_credential` |
| appid | string | 是 | 服务号唯一凭证 |
| secret | string | 是 | 服务号密钥 |

**返回示例**:
```json
{
  "access_token": "ACCESS_TOKEN",
  "expires_in": 7200
}
```

**关键特性**:
- ⏱️ **有效期**: 7200 秒（2小时）
- 🔒 **安全要求**: 必须在服务器端调用，禁止前端直接调用
- 📝 **存储建议**: 需妥善保存，避免频繁刷新
- 🆕 **稳定版 Token**: 提供 `stable_token` 接口，与原接口隔离

**重要错误码**:
| 错误码 | 说明 |
|--------|------|
| 40001 | AppSecret 错误或 Access Token 无效 |
| 40164 | 调用接口的 IP 不在白名单中 |
| 40243 | AppSecret 已被冻结 |

##### 3. 图文消息发布流程

**发布流程**: 创建草稿 → 发布草稿 → 接收事件推送

**步骤 1: 新增草稿**
```
POST https://api.weixin.qq.com/cgi-bin/draft/add?access_token=ACCESS_TOKEN
```

**请求体示例**:
```json
{
  "articles": [
    {
      "article_type": "news",
      "title": "文章标题（不超过32字）",
      "author": "作者名（不超过16字）",
      "digest": "摘要（不超过128字，单图文有效）",
      "content": "图文内容（支持HTML，少于2万字符）",
      "content_source_url": "阅读原文URL",
      "thumb_media_id": "封面图片素材ID（永久素材）",
      "need_open_comment": 1,
      "only_fans_can_comment": 0
    }
  ]
}
```

**步骤 2: 发布草稿**
```
POST https://api.weixin.qq.com/cgi-bin/freepublish/submit?access_token=ACCESS_TOKEN
```

**请求体**:
```json
{
  "media_id": "MEDIA_ID"
}
```

**返回参数**:
| 参数 | 说明 |
|------|------|
| publish_id | 发布任务 ID |
| msg_data_id | 消息数据 ID |

**步骤 3: 接收发布结果事件推送**
- 微信会推送 `PUBLISHJOBFINISH` 事件到回调 URL
- 发布状态: 0成功, 1发布中, 2原创失败, 3常规失败, 4平台审核不通过

##### 4. 模板消息限制

**接口地址**: `POST /cgi-bin/message/template/send`

**适用范围**: 仅认证服务号

**核心限制**:
| 限制项 | 说明 |
|--------|------|
| 行业设置 | 需先设置所属行业 |
| 模板选用 | 从类目模板库选用模板 |
| 参数限制 | 每个参数有严格的类型和长度限制 |
| 防重入 | 支持 client_msg_id 防重复发送 |

**参数类型限制**:
| 类型 | 格式要求 | 示例 |
|------|----------|------|
| thing.DATA | 1-20个字符 | 姓名、地址 |
| character_string.DATA | 32位以内数字/字母 | 订单号、卡号 |
| time.DATA | HH:MM 或 年月日 HH:MM | 15:01 |
| amount.DATA | 币种符号+10位数字 | ¥100 |
| phone_number.DATA | 17位以内 | +86-0766-66888866 |
| const.DATA | 20位以内，需审核 | 状态枚举 |

**重要错误码**:
| 错误码 | 说明 |
|--------|------|
| 40249 | 禁止发送营销内容 |
| 43116 | 模板被限制下发 |
| 47003 | 参数格式错误 |

##### 5. 用户管理接口

**标签管理**:

**用户基本信息**:
| 接口 | 功能 |

| 接口 | 功能 |
|------|------|
