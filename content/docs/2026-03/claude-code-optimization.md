## 问题分类体系

### 🔴 阻塞性问题
- Claude Code 无法理解需求
- 生成的代码完全无法运行
- 环境配置冲突导致开发中断
- 权限问题无法解决

### 🟡 效率问题  
- 需要多次迭代才能正确
- 上下文丢失需要重复说明
- 权限请求过于频繁打断流程
- 响应时间过长

### 🟢 质量问题
- 代码不符合项目规范
- 缺少错误处理
- 测试覆盖不足
- 性能问题

### 🔵 体验问题
- 与 git 工作流冲突
- 文件权限混乱
- 输出格式不一致
- 交互体验差
