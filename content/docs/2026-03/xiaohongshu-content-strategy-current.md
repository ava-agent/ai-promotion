## 📝 当前推荐标题模板（下周使用）

### 踩坑实录类（主推）
1. **时间+问题型**："做{项目名}，我被{技术/工具}坑了{时间}"
   - 例："做 Trip Agent，我被地图 API 坑了3天"
   
2. **预期反差型**："本来以为{简单预期}，结果{意外情况}"
   - 例："本来以为调用 API 很简单，结果踩了一堆坑"

3. **具体数字型**："{项目名}开发了{时间}，说说真实感受"
   - 例："Trip Agent 开发了8个月，说说真实感受"

### 开发故事类
1. **为什么做型**："为什么我要做{项目名}这个项目"
2. **决策过程型**："{项目名}的技术选型，我为什么选{技术}"
3. **意外发现型**："做{项目名}之后，我发现{意外发现}"

### 技术分享类
1. **实战经验型**："{技术}实战经验：{具体主题}"
2. **踩坑总结型**："{技术}踩坑总结，这几条建议能帮你省{时间}"
