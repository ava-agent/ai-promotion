## 📝 评论策略（人类化）

### 核心原则
- **口语化中文**: "哈哈"、"说实话"、"挺有意思的"
- **个人经历**: "我之前也..."、"踩过同样的坑"
- **情感表达**: emoji、语气词
- **互动提问**: "想请教..."、"你是怎么处理的？"

### 评论模板示例

**技术讨论型**:
```
这个思路挺有意思的 👍

我之前也遇到过类似的问题，当时用的是 [方法]，效果还不错。
不过你提到的 [点] 我之前没考虑过，感觉可以结合起来试试。

想请教一下，[问题] 你是怎么处理的？
```

**经验分享型**:
```
哈哈，踩过同样的坑 😂

当时我也是 [情况]，搞了 [时间] 才搞明白。最坑的是 [细节]。

后来我是 [解决方案]，虽然笨了点儿，但挺管用。

不过你的方法看起来更优雅，学习了！
```

## ⚠️ 安全限制

### 频率控制
- **评论**: 每小时 ≤3条，单日 ≤20条
- **点赞**: 每小时 ≤10个，单日 ≤50个
- **收藏**: 每小时 ≤5个
- **关注**: 每小时 ≤2个

### 随机延迟
- 操作间随机间隔 3-15 秒
- 模拟真实人类浏览行为
- 避免触发反爬虫机制
