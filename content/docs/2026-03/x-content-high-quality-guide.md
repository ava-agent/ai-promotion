---

为什么选CSP而不是贪心算法？

贪心算法快但解质量差，
用户需要的是"最优"而不是"还行"。

CSP可以：
- 定义硬约束（必须满足）
- 定义软约束（尽量满足）
- 找到全局最优解

---

代码片段：核心约束定义

```python
model = cp_model.CpModel()

# 硬约束：景点开放时间
for visit in visits:
    model.Add(visit.time >= venue.open_time)
    model.Add(visit.time <= venue.close_time)

# 软约束：用户偏好权重
model.Maximize(
    sum(visit.preference_score * visit.is_selected)
)
```
